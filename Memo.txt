git push 步驟
1. Add new repository
2. Create .gitignore
3. Initialize
    a.  cd /Users/kris/Documents/VSC/ContractBot/contractbot/ContractBot_Project
    b.  git init
    c.  git add .
    d.  git commit -m "Initial commit: Add ContractBot project files"
4. Push
    git remote add origin https://github.com/Hayamayama/Project_ContractBot.git
    git branch -M main
    git push -u origin main
[如果不行push main]，就用git rm --cached .env

git log

檢查	
git config user.name	檢查目前設定的名稱
git config user.email	檢查目前設定的信箱
全域設定
git config --global user.name "名字"	設定所有專案共用的名稱
git config --global user.email "信箱"	設定所有專案共用的信箱
專案設定	
git config user.name "名字"	只設定當前專案的名稱
git config user.email "信箱"	只設定當前專案的信箱
